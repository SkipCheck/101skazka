\# 101 сказка



Сайт-визитка для компании, предоставляющей услуги, связанные со сказками и детскими мероприятиями.

![Главная страница](screenshots/homepage.png)
![Услуги](screenshots/services.png)
![Форма заказа](screenshots/order-form.png)


\*\*101 сказка\*\* — это многостраничный веб-сайт, разработанный в качестве пет-проекта для демонстрации навыков веб-разработки. Проект включает в себя:



\- Формы обратной связи и заказа услуг с записью данных в базу данных

\- Структурированные разделы: главная страница, услуги, новости, о нас, контакты

\- Административную панель (папка `\_admin`)



\## Технологии



\- \*\*Backend:\*\* PHP

\- \*\*Frontend:\*\* HTML, CSS, JavaScript

\- \*\*База данных:\*\* MySQL (дамп базы — `skazkadb.sql`)

\- \*\*Дополнительно:\*\* Hack (5.7% кодовой базы)
